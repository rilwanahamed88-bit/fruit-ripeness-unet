# 🍎 Fruit Ripeness Classification Using U-Net Model

## 📌 Project Overview

Fruit quality assessment is an important process in agriculture and food industries to maintain product quality and reduce crop losses. This project presents an automated Fruit Ripeness Classification System using the U-Net Deep Learning Model.

The system classifies fruits into:

- 🟢 Normal
- 🟡 Unripe
- 🟠 Ripened
- 🔴 Affected

It also provides fertilizer recommendations when affected fruits are detected.

---

## 🎯 Objectives

- Develop an automated fruit quality classification system.
- Apply U-Net architecture for image segmentation.
- Classify fruits based on their condition.
- Reduce manual inspection effort.
- Improve classification accuracy.
- Detect affected fruits at an early stage.
- Provide fertilizer recommendations.
